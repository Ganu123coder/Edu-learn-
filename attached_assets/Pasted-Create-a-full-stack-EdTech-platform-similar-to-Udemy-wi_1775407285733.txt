Create a full-stack EdTech platform similar to Udemy with an integrated AI Learning Planner. The platform should support three main roles: Admin, Trainer, and Student. Trainers can create courses, students can enroll and track learning, and the AI system helps students plan their learning path.

Tech Stack:
Frontend: React.js with Tailwind CSS
Backend: Node.js with Express
Database: MongoDB (Mongoose)
Authentication: JWT + bcrypt
AI Integration: OpenAI API (or any LLM API)

---

1. Authentication System

Implement secure authentication and role-based access.

Features:

* User registration and login
* Role selection during signup (Student or Trainer)
* Admin account manually created in database
* Password hashing using bcrypt
* JWT-based authentication
* Protected routes for each role

User Model Fields:

* name
* email
* password
* role (admin, trainer, student)
* profilePhoto
* createdAt

---

6. AI Learning Planner Feature

Create an AI-powered system that generates personalized learning plans.

AI Planner Input Form:
Students should enter:

* Career goal (Example: Data Scientist, Full Stack Developer)
* Current skill level (Beginner, Intermediate, Advanced)
* Available study hours per day
* Learning duration (30, 60, 90 days)
* Preferred technologies
* Learning style (videos, projects, theory)

When the form is submitted:
Send the input to backend AI API.

AI should generate:

* Personalized learning roadmap
* Weekly study schedule
* Recommended platform courses
* Project suggestions
* Estimated completion timeline

Example AI Output:

Week 1–2
Learn HTML and CSS fundamentals
Recommended Course: Web Development Basics

Week 3–4
JavaScript Fundamentals
Recommended Course: JavaScript Bootcamp

Week 5–6
React Basics
Recommended Course: React for Beginners
